Fragmentos pequenos de código (shell, YAML, configs) prontos para copiar em contextos específicos. Diferente de recipes de comando (que são processos), snippets são blocos de código que se encaixam em estruturas maiores.

## Por linguagem/formato

- **Bash** — funções, loops, padrões de erro-handling.
- **Fish shell** — aliases, funções, configuração.
- **Systemd** — units, timers, targets.
- **Docker Compose** — serviços, redes, volumes comuns.
- **Kubernetes** — manifests, labels, selectors.
- **Helm** — values.yaml, templates, Charts.
- **Kustomize** — patches, overlays, bases.
- **Argo CD** — applications, sync policies.
- **Traefik** — middleware, services, rotas.
- **Prometheus** — scrape configs, alerting rules.
- **GitHub Actions** — workflows, jobs, steps.
- **GitLab CI** — pipelines, jobs, stages.
